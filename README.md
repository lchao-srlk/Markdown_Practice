## 📂 仓库内容

### 1️⃣ `practice.md` —— Markdown 语法练习手册
这份文件涵盖了 **Markdown 的绝大多数常用语法**，包括：
- 标题层级（H1 ~ H6）
- 引用、列表（有序/无序/任务清单）
- 代码块（多种语言高亮）
- 数学公式（行内与块级）
- 表格、脚注、横线
- 链接、图片、引用链接
- 文本修饰（斜体、粗体、删除线、下划线、高亮）
- 表情符号（基础版）和行内 HTML

> 这份练习是作者跟着B站某博主的视频学习时自己练手用的，适合边看边动手敲一遍，仅供参考。原视频链接见文末。
